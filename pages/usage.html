Usage guidelines

Disclaimer

LeapcapJS is still a proof of concept and was made in a limited timeframe. It will be curated in the coming months to be more usable, and more stable. Howevever, you can still use the current build, with its current features. As always, please collaborate on GitHub!

Usage

Add the necessary libraries to your HTML

    	<script>libs/leap.min.js</script>
    	<script>libs/three.min.js</script>
    	<script>libs/leapcap.js</script>

Prepare where the renderer should be 

    	<style>.my_canvas{width:800px;height:950px;}</style>
    	<div class="my_canvas"></div>
The WebGL renderer uses the size of your element.

Initialize LeapCapJS with one line

    	//jQuery is not a dependency, but you may still use it to pass elements
    	LeapCap.initLeapCap({'scene':$('.my_canvas')});

Access LeapCap methods

    	LeapCap.record() //starts to capture frames
		LeapCap.play() //plays back captured frames
		LeapCap.play(savedFrames) //plays back a saved array of captured frames, retrieved using LeapCap.getSavedFrames()
		LeapCap.clearFrames() //deletes captured frames
		LeapCap.getSavedFrames() //returns an array of saved motion data.
